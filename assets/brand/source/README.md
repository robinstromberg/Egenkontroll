# Godkända masterfiler för varumärket

`Icon.svg` och `Logo.svg` är de godkända masterkällorna för Min Egenkontrolls kommande visuella identitet.

De ligger medvetet utanför `public/` och används ännu inte av produktionen. Inga befintliga logotyper, ikoner, manifest, faviconer, PWA-filer eller andra aktiva brandreferenser ska ändras som en del av denna leverans.

Ändra inte, optimera inte och exportera inte om masterfilerna här. En framtida, avgränsad brandimplementering ska utgå från dessa källor för att skapa nödvändiga varianter och uppdatera aktiva användningsställen konsekvent.
