Allergenöverföring

Hur minskar man risken för allergenkontamination?

Allergena ämnen kan överföras mellan livsmedel via redskap, utrustning, förvaring och felaktig märkning. Rutinerna behöver därför täcka hela hanteringen.

Rengör och kontrollera

Utrustning som använts för allergena livsmedel behöver rengöras och kontrolleras innan den används för andra produkter.

Förvara åtskilt

Allergena råvaror kan behöva hållas separerade från andra livsmedel.

Märk internt

Färdiga rätter och ingredienser utanför originalförpackning kan behöva tydlig intern märkning för att undvika förväxling.

Räcker det att personalen kan svara på frågor?

Nej. Korrekt information är viktig, men verksamheten behöver också så långt som möjligt förhindra att allergena ingredienser överförs från ett livsmedel till ett annat.
