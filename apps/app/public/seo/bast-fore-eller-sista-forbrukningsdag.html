Datummärkning

Bäst före eller sista förbrukningsdag – vad är skillnaden?

De två datummärkningarna betyder olika saker. Bäst före handlar främst om hur länge livsmedlet behåller sina egenskaper, medan sista förbrukningsdag används för vissa mycket lättfördärvliga livsmedel som efter kort tid kan innebära omedelbar hälsorisk.

Bäst före

Anger minsta hållbarhet när produkten förvarats på lämpligt sätt.

Sista förbrukningsdag

Används för vissa mycket lättfördärvliga livsmedel där lagringstiden kan påverka säkerheten.

Företaget ansvarar

Den som märker eller säljer produkten ansvarar för att rätt märkning används.

Är sista förbrukningsdag bara ett "strängare bäst före"?

Nej. Skillnaden är principiell. Efter bäst före kan ett livsmedel fortfarande få säljas om företaget bedömer att det är säkert. Efter sista förbrukningsdag betraktas livsmedlet som icke säkert och får inte säljas eller skänkas.
