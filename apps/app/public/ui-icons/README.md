# UI icons

Lägg bildfiler som används som UI-ikoner här.

Exempel:

- ikoner för kontrolltyper
- ikoner för knappar
- små illustrationsikoner som används i appens gränssnitt

Rekommenderad namngivning:

- använd små bokstäver
- använd bindestreck
- undvik å, ä och ö i filnamn

Exempel:

- `kyltemperatur.png`
- `stadning.png`
- `sparbarhet.png`
- `varumottagning.png`
- `allergener.png`

Placera inte brand/logotyp-filer här. De hör hemma i `public/brand/`.
